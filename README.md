# Springboard Learnings

This repo is a record of everything I worked through during my Springboard Software Engineering bootcamp. It goes from the very basics of HTML all the way to a fully routed React app, and it is organized the same way the curriculum is, so you can follow the progression from start to finish.

I am keeping this here because deleting it would mean losing the work, and the work actually tells a story worth keeping.

## What is in here

**HTML and CSS** is where everything started. Basic page structure, semantic HTML, tables, forms, and a practice portfolio page built with raw HTML and CSS.

**JavaScript** covers the full language track: core JS fundamentals, intermediate topics like async programming, ES6+ features, and data structures like trees and graphs, browser JS with the DOM and events and localStorage, and API work with REST, GraphQL, and an MVC-based Jeopardy game.

**Building View, React.js** is the biggest section. It covers everything from writing a first component all the way to building a multi-page job board app with React Router, nested layouts, breadcrumbs, form actions, and error handling.

## What is not in here

The Capstone Project lives in its own repo. The Git and GitHub practice folders are not included since they are just git workflow exercises with text files, not real code.

## Tech used

HTML, CSS, JavaScript, React, Vite, React Router, Axios, and a handful of public APIs like the Deck of Cards API, PokéAPI, and the GitHub API.
