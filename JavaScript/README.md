# JavaScript

This covers everything I worked through in JavaScript, from the very basics all the way to async programming, data structures, DOM manipulation, and APIs. It is broken into four tracks that build on each other.

## Basic JavaScript

This is where I got comfortable with the core language. Topics include primitive data types (strings, numbers, booleans), program logic and conditionals, arrays, objects, loops, and functions. The exercises here are small and focused, each one targeting a specific concept.

## Intermediate JavaScript

This section went deeper into the language. Here is what it covers.

**Error handling and debugging** covered how to catch exceptions cleanly and write more resilient code.

**Sorting** covered different sorting algorithms and how JavaScript handles comparison under the hood.

**Async programming** was a big one. This covers callbacks, Promises, and async/await. There are two exercises here: one working with a Pokemon API and another that draws cards from a deck API.

**ES6 features** are split into a few sub-folders covering arrow functions, hoisting, rest and spread operators, destructuring (arrays and objects), and JavaScript modules with import and export.

**Advanced data structures** covers binary trees, binary search trees, and graphs, including how to traverse and work with them in code.

## JavaScript and the Web

This section is about making JavaScript interact with the browser.

**The Document Object Model** covers selecting elements, changing content, creating nodes, and modifying the DOM dynamically.

**Events** covers click handlers, form submission events, keyboard events, and event delegation.

**Storing data in browsers** covers localStorage, including how to save, read, and remove data so it persists between page visits.

## JavaScript and APIs

This section connects JavaScript to external data.

**REST and RESTful APIs** covers how HTTP requests work, how to use fetch, and how to handle responses. There is a small app that pulls data from a public API.

**GraphQL** is a short intro covering the difference between GraphQL and REST, and how to write basic queries.

**MVC and Jeopardy** is the most complete project in this section. It is a Jeopardy game built with a model, view, controller pattern, pulling questions from the jService API.

## What I learned

By the end of this section I could write clean JavaScript, work with async data from external APIs, manipulate the DOM without a framework, and understand how to structure code using patterns like MVC and modules.
