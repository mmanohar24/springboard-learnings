# HTML and CSS

This is where I started learning the fundamentals of building things on the web. Nothing fancy, just getting comfortable with how HTML and CSS actually work before moving into JavaScript.

## What is in here

**3.2 HTML Fundamentals** is the first exercise set. It covers the basics like structuring a page with headings, paragraphs, links, images, and semantic tags. There is a small demo site about sequoia trees that I built just to practice putting elements together.

**3.3 Tables and Forms** picks up where that left off. Tables were straightforward once I understood the thead, tbody, and tfoot structure. Forms took a bit more time since there are a lot of input types to learn, like text, email, radio, checkboxes, selects, and textareas.

**HTML Practice** is a freeform file where I experimented outside of the exercises. I built a rough portfolio layout in HTML and CSS to practice what I learned, applying real structure and styling to something that looks like an actual page.

## What I learned

By the end of this section I could build a static webpage from scratch, structure it with semantic HTML, style it with CSS, and build a working contact form. It also gave me a solid mental model of how browsers read and render HTML before JavaScript gets involved.
